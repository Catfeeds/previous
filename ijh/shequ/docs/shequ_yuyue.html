订座下单(client/yuyue/dingzuo/create)

请求参数

参数名称 是否必须 类型 描述
shop_id 是 int 商户ID
yuyue_time 是 datetime 预定来店时间
dingzuo_number 是 int 就餐人数
is_baoxiang 否 int 是否需要包厢
contact 是 string 联系人
mobile 是 string 手机号
notice 否 string 预定备注

请求示例

{
    "shop_id" : "1",
    "yuyue_time" : "2016-10-08 18:00",
    "dingzuo_number" : "10",
    "is_baoxiang" : "1",
    "contact" : "游医",
    "mobile" : "1388888888",
    "notice" : "订单备注"
}

返回数据

参数名称 类型 描述

    }
}
